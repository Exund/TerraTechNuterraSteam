Build instructions

1. Copy the "Mods/NuterraSteam" subfolder into your TerraTech Official Mod Tool installation in "Assets/Mods/"
2. Open the VS project
3. Ensure that all the required references are set up correctly. (You can find all the required DLLs in your TerraTech install directory then /TerraTech_Data/Managed)
4. Press Ctrl+Shift+B to build
5. Copy the output DLL from "Built" folder to the NuterraSteam folder in your Unity project.
6. Upload to Steam (If you need permissions, ask the Payload Community Manager on Discord)
